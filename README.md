# mzegar.github.io

Backup of my personal portfolio website created using React. 

## Project Structure

- src/...
  - main.js - Main webpage file
  - src/components/...
    - body.js - Body for the webpage
    - header.js - Header for the webpage
    - navbar.js - Navigation bar for the webpage
    
## Building

`npm install`
    
## Deployment

`yarn run deploy`
